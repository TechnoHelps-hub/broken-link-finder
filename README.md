# Website Broken Link Finder & URL Status Checker 🔍

A fast, lightweight Python script designed to crawl any website, scan all its internal hyperlinks, and instantly identify broken links (`404 Not Found`, `500 Server Error`, etc.). This tool helps webmasters, bloggers, and SEO professionals clean up dead links, improve crawl efficiency, and significantly boost search engine rankings.

## ✨ Features
- **Deep Internal Crawling:** Automatically crawls internal pages starting from your homepage.
- **HTTP Status Code Tracking:** Instantly reports standard web codes (`200 OK`, `404 Broken`, `301 Redirect`).
- **Dead Link Filter:** Clearly highlights broken links at the end of the report for quick fixes.
- **Smart Queueing:** Avoids crawling duplicate URLs to optimize speed and resource usage.

## 🚀 Quick Start Guide

### 1. Prerequisites
Make sure you have Python installed, then install the required `requests` and `beautifulsoup4` libraries:
```bash
pip install requests beautifulsoup4

2. How to Use

    Clone this repository or download the script.

    Open the script file and replace the target URL with your website address.

    Run the script:

Bash

python link_checker.py

    Check the terminal output for a clean, live report of your site's link health!
